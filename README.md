# 🎓 Cando Education
**Real-time Educational Game Platform**

Cando Education is a multiplayer classroom tool inspired by Gimkit. Teachers can create question kits, host live game sessions with 6-digit join codes, and students can join with custom names and 2D chibi avatars to compete in real-time.

---

## 🚀 Features
* **Live Game Lobby:** Teachers generate unique codes; students join instantly via WebSockets.
* **Custom Avatars:** Support for Google Auth-linked chibi characters.
* **Question Kits:** Teachers can build and share sets of educational questions.
* **Koyeb Ready:** Configured for easy deployment to the cloud.

---

## 🛠️ Tech Stack
* **Frontend:** React.js, Socket.io-client
* **Backend:** Node.js, Express.js, Socket.io
* **Database:** MongoDB (Planned)
* **Game Engine:** Phaser.js (Planned for 2D movement)
